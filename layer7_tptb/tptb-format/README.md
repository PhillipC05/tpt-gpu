# tptb-format

TPT Script formatter and linter — canonical pretty-printer and style enforcer for `.tpts` source files.

## Overview

`tptb-format` consumes a `tptb-core` AST and re-emits normalized TPT Script source. It is the backend for the `tpt fmt` subcommand and the LSP formatting provider.

## Usage

```toml
[dependencies]
tptb-format = "1.0"
```
